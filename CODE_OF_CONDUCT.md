# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — filing issues, opening pull requests, posting in discussions, etc. — you agree to abide by its terms.

## Reporting

If you see or experience behavior that violates the Contributor Covenant, email **erenakbulutwork@gmail.com** with the subject prefix `Endless Canvas CoC:`. Reports will be handled privately.

For security-related concerns (e.g. disclosure of operator keys), see [`SECURITY.md`](SECURITY.md) instead.
